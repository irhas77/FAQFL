+

Bagaimana jika saya tidak bisa log in karena lupa password?

+

Jika Anda terkendala lupa password dan reset password, silakan menghubungi Biro Teknologi Informasi (BTI) melalui laman: http://support.mercubuana.ac.id, lalu pilih menu Open a new ticket, pada help topic pilih Fast Learning / Akses (log in) lalu isi datanya sesuai dengan data Anda. Setelah itu tim IT akan mengirimkan link untuk reset password melalui email.

Atau hubungi hotline BTI UMB: +62 851-3337-0871

Menu apa saja yang terdapat pada tampilan dashboard Fast Learning?

+

Pada tampilan dashboard Fast Learning, terdapat beberapa menu dengan masing-masing fungsinya. Berikut adalah penjelasannya:

  • Home: Halaman awal Fast Learning, pada halaman ini berisi pengumuman, panduan penggunaan Fast Learning.
  • My Course: Mata kuliah yang Anda ikuti pada semester berjalan.
  • Profile: Informasi tentang profil Anda & menu untuk ubah password.

Contoh email student: 4145883897@student.mercubuana.ac.id
